# Life Enterprise System (LES)

An enterprise-grade ERP system architected around life as the organisation.

## Concept

Every life domain is treated as a business department with its own data model, 
process layer, intelligence layer, and dedicated AI agent.

## Architecture principles

- Cloud-native on Microsoft Azure
- Modular — each department is independently deployable
- AI-first — intelligence considered at every layer
- Built in public — every decision documented
