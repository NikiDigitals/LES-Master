## Core Parameters — Read These First

**Target:** Finance Transformation Architect — finance systems at full depth including technology architecture, IFRS, management accounting, international taxation, financial law, AML, corporate finance, and AI-augmented financial intelligence.

**Depth standard:** Global expert. Not practitioner level. Not exam pass level. The deepest possible understanding that exists. This is ADR-000066 — permanent and non-negotiable.

**Methodology:** Five steps — Study, Discuss, Sandbox, Review, Apply. No concept implemented in LES until Step 4 complete.

**Quarter convention:** Q1 = July–September. Q2 = October–December. Q3 = January–March. Q4 = April–June.

## What Claude Must Never Do in Education Sessions

- Give minimum viable answers to learning questions. The depth standard is global expert. Always.
- Suggest skipping a step in the five-step learning methodology.
- Recommend a certification not in the pathway without explicitly framing it as a proposal.
- Assume the doctoral question is open. It is closed. Not pursuing. Final.
- Design around the formal programmes without checking TMA and exam deadlines first.
- Produce GitBook content autonomously. Only when explicitly requested.
- Decide which ACCA paper comes next without checking the sequence table above.
- Suggest a Python dedicated track — Python is embedded in TM112 and Masters in AI.
- Suggest a separate IFRS track — IFRS is embedded in ACCA sequence.
- Reference CompTIA ITF+ or ISC2 CC — both removed from the programme.

## Session Close Protocol — Education Sessions

At the close of every LES-LEARN session Claude produces:

1. **Concepts confirmed understood** — what was discussed and verified
2. **Sandbox assigned** — what to build to test the understanding
3. **LES application identified** — where this concept applies in LES
4. **ADRs to log** — any decisions made during the session
5. **Next session recommended** — what the logical next topic is

At the close of every LES-MENTOR session Claude produces the standard documentation briefing including education-specific items:

1. **ADRs to write** — all decisions from this session
2. **Current status table update** — which row needs updating
3. **Quarterly document updates** — if any decisions change a quarterly plan
4. **Journal entry** — education progress to include in weekly journal
